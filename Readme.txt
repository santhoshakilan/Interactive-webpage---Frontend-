Project Description:
                   This project is a complete front-end web application that combines all
eight tasks from the Cognifyz internship into one functional webpage.


The project includes:
--------------------

1. Basic HTML Structure  
2. Inline CSS Styling  
3. Responsive Design using Media Queries  
4. Interactive Button using JavaScript  
5. API Integration using Fetch()  
6. Form Styling and Client-Side Validation  
7. Component-Based Styling using Bootstrap  
8. CSS Preprocessing Concept (Sass Variables & Nested Style Structure)

All features are implemented in:
- index.html
- styles.css
- script.js

         ----------------------------

How to Run the Project:
----------------------
1. Download the project folder.
2. Open "index.html" in any web browser (Chrome recommended).
3. Ensure "styles.css" and "script.js" are in the same folder as index.html.
4. You can also run it using Live Server in VS Code.
